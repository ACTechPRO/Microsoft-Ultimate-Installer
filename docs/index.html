Como Usar

3 Passos Simples

1

Abra o PowerShell

Clique direito no menu iniciar e selecione "Windows PowerShell (Admin)" ou "Terminal (Admin)".

2

Cole o Comando

Copie o comando de instalação acima e cole no PowerShell. Pressione Enter.

3

Escolha as Opções

A interface gráfica abrirá. Selecione o que deseja instalar e clique em Executar.
